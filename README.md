# ecommerce-frontend-website
A responsive E-Commerce frontend website built using HTML, CSS, and JavaScript featuring product listing, shopping cart functionality, and total price calculation.

A responsive E-Commerce frontend website built using HTML, CSS, and JavaScript.

## 🚀 Features

* Product listing
* Add to cart
* Remove from cart
* Auto total price calculation
* Responsive layout

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## ▶️ How to Run

Open `index.html` in your browser.
